
## Description

API for the challenge [Image Uploader](https://devchallenges.io/challenges/O2iGT9yBd6xZBrOcVirx) presented in [devChallenges.io](https://devchallenges.io/).


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
